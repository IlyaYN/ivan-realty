# 🏡 Шаблон сайта недвижимости (Next.js + Google Sheets)

Современный, быстрый и безопасный сайт для агента по недвижимости. 
Проект использует **Google Таблицы** в качестве удобной и бесплатной CMS (базы данных) и **Telegram API** для моментального получения заявок без блокировок.

## 🚀 Технологии

* **Фреймворк:** Next.js 15 (App Router) / React
* **Стилизация:** Tailwind CSS / CSS Modules
* **База данных / Контент:** Google Sheets API (формат CSV)
* **Интеграции:** Telegram Bot API (серверная отправка заявок), VK API (Парсинг постов и витрины)
